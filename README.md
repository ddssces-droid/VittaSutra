# VittaSutra

## Version
V1.0

## Objective
ATH Based Trading System for NSE Stocks.

## Scanner
- ATH
- DCC
- WCC
- MCC
- RATH

## Position Building
- DCC = 30%
- WCC = 30%
- MCC = 40%

## Profit Booking
T1 = +12.5%
T2 = +25%
T3 = +50%
T4 = +75%
T5 = +100%

## Stop Loss

Before T5:
MAX(Avg Entry -25%, Previous Month 20 SMA)

After T5:
MAX(Avg Entry, Previous Month 10 SMA)
